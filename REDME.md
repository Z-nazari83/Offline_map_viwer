# Offline Location Map Viewer

## توضیح پروژه
این پروژه یک برنامه پایتونی است که امکان "جستجوی مکان بر اساس نام" و نمایش آن روی نقشه را فراهم می‌کند.

فرآیند کار به این صورت است:
- در "اجرای اول"، برنامه با استفاده از اینترنت و API سرویس OpenStreetMap (Nominatim)،
  مختصات جغرافیایی (Latitude و Longitude) مکان واردشده را دریافت می‌کند و در پایگاه داده SQLite ذخیره می‌نماید.
- در "اجرای‌های بعدی"، اگر مکان قبلاً در دیتابیس ذخیره شده باشد،
  برنامه بدون نیاز به اینترنت، اطلاعات را از دیتابیس خوانده و نقشه ذخیره‌شده را اجرا می‌کند.

هدف این پروژه، کاهش وابستگی به اینترنت و افزایش سرعت اجرای برنامه با استفاده از کش‌کردن داده‌ها است.

## امکانات پروژه
- ذخیره مختصات مکان‌ها در دیتابیس SQLite
- استفاده مجدد از داده‌های ذخیره‌شده
- نمایش مکان روی نقشه با استفاده از Folium
- جلوگیری از ذخیره‌ی تکراری نقشه‌ها
- باز کردن خودکار نقشه در مرورگر
